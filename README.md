# Sales Insights Analysis using Python \& SQL

## Project Overview

This project focuses on analyzing structured sales data to extract meaningful business insights using Python and SQL. It demonstrates real-world data handling, analysis, and visualization techniques suitable for entry-level roles in Python and data analytics.



Tech Stack

* Python
* Pandas, NumPy
* Matplotlib
* SQLite

## Project Structure

sales-data-analysis/
├── data/
│   └── sales.csv
├── src/
│   ├── main.py
│   ├── analysis.py
│   └── database.py
├── output/
│   └── monthly\_sales.png
├── README.md
└── requirements.txt

## Features

* Performed data cleaning and preprocessing using Pandas to ensure data accuracy and consistency
* Analyzed revenue trends and generated product-level insights to identify top-performing items
* Developed visualizations of monthly sales trends using Matplotlib for better business understanding
* Executed SQL queries to evaluate region-wise revenue performance and distribution
* Designed a modular and scalable codebase for improved readability and maintainability



## Insights Generated

* Calculated total revenue to evaluate overall business performance
* Identified top-performing products driving maximum sales and profitability
* Analyzed monthly sales trends to uncover seasonal patterns and growth opportunities
* Assessed region-wise revenue contribution to highlight high-performing markets

## How to Run

1. Install dependencies
pip install -r requirements.txt
2. Run the project
python src/main.py
3. Output
* Console displays sales insights
* Sales trend chart saved in output/monthly\_sales.png
